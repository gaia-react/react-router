# FontAwesome

Icon set. GAIA includes `free-solid`, `free-regular`, `free-brands`, plus `react-fontawesome` and the SVG core.

`config.autoAddCss = false` is set in `root.tsx` to prevent FontAwesome from injecting its own CSS (Tailwind handles sizing).

## Alternatives

[Heroicons](https://heroicons.com/), [Feather](https://feathericons.com/), Lucide, etc.
